API 키 발급 가이드

YouTube Script Mate에서 사용할 수 있는 AI 서비스 API 키 발급 방법

G
Google Gemini AI 번역 + 요약

Google의 최신 AI 모델. 무료 티어 제공으로 가볍게 시작하기 좋음.

1
Google AI Studio에 접속하여 Google 계정으로 로그인
2
"Create API Key" 버튼 클릭
3
생성된 API 키를 복사하여 설정에 입력

추천 모델명

gemini-2.5-flash gemini-2.5-flash-lite gemini-1.5-pro
무료 티어: 분당 15 요청, 일일 1,500 요청 제공
O
OpenAI (ChatGPT) AI 번역 + 요약

GPT-4o, GPT-3.5 등 ChatGPT 모델 사용. 정확도 높은 번역과 요약 제공.

1
OpenAI Platform에 가입
2
API Keys 페이지에서 "Create new secret key" 클릭
3
생성된 API 키를 복사하여 설정에 입력

추천 모델명

gpt-4o gpt-4o-mini gpt-3.5-turbo
유료: 사용량에 따라 과금. gpt-4o-mini는 저렴하고 빠름 ($0.15/1M input tokens)
A
Anthropic (Claude) AI 번역 + 요약

Claude AI 모델. 긴 문맥 이해에 뛰어나며 자연스러운 번역 품질.

1
Anthropic Console에 가입
2
Settings → API Keys 메뉴에서 "Create Key" 클릭
3
생성된 API 키를 복사하여 설정에 입력

추천 모델명

claude-sonnet-4-20250514 claude-3-5-haiku-20241022
유료: 사용량에 따라 과금. claude-3-5-haiku가 가장 저렴 ($0.25/1M input tokens)
D
DeepL 번역 전용

전문 번역 서비스. 뉘앙스가 자연스러운 고품질 번역으로 유명.

1
DeepL API 페이지에서 "무료로 가입" 클릭
2
계정 생성 후 계정 설정 → API 탭에서 인증 키 확인
3
API 키를 복사하여 설정에 입력 (모델명은 입력 불필요)
DeepL은 번역 전용 서비스입니다. 요약 기능을 사용하려면 AI 기반 제공자를 선택하세요.
무료 티어: 월 50만 자 번역 제공. Pro 버전은 무제한
P
Naver Papago 번역 전용

네이버의 번역 서비스. 한국어 번역에 특화되어 있음.

1
Naver Developers에 로그인
2
Application → 애플리케이션 등록에서 "Papago 번역" API 선택
3
등록 후 Client ID와 Client Secret 확인
4
설정에 Client ID|Client Secret 형식으로 입력
예: abc123|xyz789
Papago는 번역 전용 서비스입니다. API 키는 "Client ID|Client Secret" 형식으로 입력하세요.
무료 티어: 일 10,000자 번역 제공
Az
Microsoft Azure OpenAI AI 번역 + 요약

기업용 Azure 클라우드에서 제공하는 OpenAI 모델. 보안과 안정성 중시.

1
Azure Portal에서 Azure OpenAI 리소스 생성
2
Azure OpenAI Studio에서 모델 배포
3
리소스의 엔드포인트 URL과 API 키 확인
4
설정에 엔드포인트URL|API키 형식으로 입력
예: https://myresource.openai.azure.com|abc123xyz

모델명 (배포 시 설정한 이름)

gpt-4o gpt-35-turbo
유료: Azure 사용량에 따라 과금. 기업용 보안 및 규정 준수 기능 제공
C
Cohere AI 번역 + 요약

다국어 지원이 뛰어난 AI 모델. 무료 평가판 제공.

1
Cohere Dashboard에 가입
2
API Keys 메뉴에서 "Create Trial Key" 또는 "Create Production Key" 클릭
3
생성된 API 키를 복사하여 설정에 입력

추천 모델명

command-r-plus command-r
Trial 키: 무료로 테스트 가능. Production 사용 시 과금
M
Mistral AI AI 번역 + 요약

유럽 기반의 오픈소스 AI 모델. 빠른 속도와 효율적인 성능.

1
Mistral Console에 가입
2
API Keys 메뉴에서 "Create new key" 클릭
3
생성된 API 키를 복사하여 설정에 입력

추천 모델명

mistral-large-latest mistral-medium-latest mistral-small-latest
유료: 사용량에 따라 과금. mistral-small이 가장 저렴 (€0.1/1M tokens)
P
Perplexity AI AI 번역 + 요약

검색 기반 AI 모델. 최신 정보와 정확한 답변 제공.

1
Perplexity Settings에 로그인
2
API Keys 섹션에서 "Generate" 버튼 클릭
3
생성된 API 키를 복사하여 설정에 입력

추천 모델명

sonar sonar-pro
유료: Pro 구독 또는 API 크레딧 필요. sonar: $5/1000 requests

3
생성된 API 키를 복사하여 설정에 입력

llama-3.3-70b-versatile llama-3.1-8b-instant mixtral-8x7b-32768
무료 티어: 분당 30 요청, 일일 14,400 요청 제공. 초고속 응답이 특징
