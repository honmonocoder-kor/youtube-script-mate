YouTube Script Mate 사용 가이드

영어 학습을 위한 유튜브 스크립트 도우미

단축키

Space 재생/일시정지 ← 5초 뒤로 (설정에서 변경 가능) → 5초 앞으로 PageUp 이전 페이지 PageDown 다음 페이지 R 구간반복 (시작/종료, 설정에서 변경 가능) Esc 녹음 종료 / 구간반복 해제
액션 버튼 단축키 (일반/번역 모드)
A: 녹음 | S: 원문 재생 | D: 녹음파일 재생 | F: 비교재생 | G: 저장

3가지 모드

  • 일반 스크립트 보기 + 쉐도잉 연습 + 표현 저장
  • 번역 AI 번역 표시 + 문맥 기반 단어 번역
  • 요약 AI 요약 생성 + 요약 내용 저장

주요 기능

  • 스크립트 추출: 새로고침 버튼으로 유튜브 자막 불러오기
  • 구간반복: 텍스트 드래그 후 R키로 선택 구간 반복
  • 쉐도잉 녹음: 문장별 녹음 후 원문과 비교 재생
  • 표현 저장: 중요 문장을 저장하고 보관함에서 복습
  • 단어 번역: 저장된 표현에서 단어 드래그 시 문맥 기반 번역

번역/요약 설정

  • API 키: Gemini, OpenAI, Claude 등 AI API 키 입력
  • 모델명: 사용할 AI 모델 지정 (예: gemini-2.5-flash)
  • 언어 설정: 스크립트 언어와 번역 대상 언어 설정
설정 아이콘(⚙)을 클릭하여 API 키를 입력하세요. 무료 티어가 있는 Gemini 추천!

사용 팁

  • 스크립트 라인 클릭 시 해당 시간으로 영상 이동
  • 배속 조절로 느린 발음 연습 가능 (+/- 버튼)
  • 저장된 표현은 영상별, 전체 보관함에서 확인
  • CSV 다운로드로 저장된 표현 백업 가능
