# YouTube Script Mate - Chrome Extension

YouTube 영상의 스크립트(자막)를 활용한 외국어 학습 도구입니다. 스크립트를 추출하고, AI를 통해 번역 및 요약하며, 학습한 표현을 저장하고 관리할 수 있습니다. 쉐도잉 연습을 위한 녹음 기능도 제공합니다.

## 주요 기능

### 스크립트 학습
- **YouTube 스크립트 추출**: YouTube 영상 페이지에서 자동으로 스크립트(자막)를 추출
- **페이지네이션**: 스크립트를 페이지별로 나누어 보기 편하게 표시
- **클릭하여 이동**: 스크립트 라인을 클릭하면 해당 시간대로 영상 이동
- **구간 반복**: 특정 구간을 반복 재생하여 집중 학습

### 3가지 학습 모드
- **일반 모드**: 스크립트 보기와 표현 저장
- **번역 모드**: AI 기반 실시간 번역과 저장
- **요약 모드**: 전체 영상 내용을 AI가 요약

### AI 번역 & 요약
- **다중 AI 제공자 지원**: Google Gemini, OpenAI(ChatGPT), Anthropic(Claude), Cohere, Mistral AI, Perplexity AI, Groq
- **전문 번역 서비스 지원**: DeepL, Naver Papago
- **Azure OpenAI Service 지원**: 기업용 Azure 클라우드 환경 지원
- **자동 제공자 감지**: 모델명 입력 시 자동으로 AI 제공자 감지
- **문맥 기반 번역**: 전후 문장을 고려한 자연스러운 번역

### 쉐도잉 연습
- **녹음 기능**: 원어민 음성을 따라 읽고 녹음
- **비교 재생**: 원문과 내 녹음을 비교하여 발음 교정
- **녹음 파일 관리**: 녹음한 파일을 저장하고 관리

### 보관함
- **표현 저장**: 학습한 표현을 영상별로 저장
- **전체 보관함**: 모든 저장된 표현을 한눈에 관리
- **CSV 내보내기**: 저장된 표현을 CSV 파일로 내보내기

## 데이터 저장 안내

**현재 버전에서는 모든 데이터(저장된 표현, 녹음 파일, 설정)가 로컬 브라우저에 저장됩니다.**

- 장점: 인터넷 연결 없이도 저장된 데이터 접근 가능, 개인정보 보호
- 주의: 브라우저 데이터 삭제 시 저장된 내용도 삭제됨

향후 업데이트에서 클라우드 동기화 기능을 추가하여, 여러 기기에서 학습 내용을 동기화할 수 있도록 지원할 예정입니다.

## 설치 방법

### Chrome 웹 스토어 (권장)
1. Chrome 웹 스토어에서 "YouTube Script Mate" 검색
2. "Chrome에 추가" 버튼 클릭

### 개발자 모드 설치
1. 이 저장소를 클론 또는 다운로드
2. Chrome에서 `chrome://extensions/` 열기
3. 우측 상단 "개발자 모드" 활성화
4. "압축해제된 확장 프로그램을 로드합니다" 클릭 후 폴더 선택

## API 키 설정

YouTube Script Mate는 다양한 AI 서비스를 지원합니다. 원하는 서비스의 API 키를 발급받아 설정에서 입력하세요.

### 지원하는 AI 제공자

| 제공자 | 기능 | 무료 티어 |
|--------|------|-----------|
| Google Gemini | 번역 + 요약 | O (분당 15 요청) |
| OpenAI (ChatGPT) | 번역 + 요약 | X |
| Anthropic (Claude) | 번역 + 요약 | X |
| Mistral AI | 번역 + 요약 | X |
| Perplexity AI | 번역 + 요약 | X |
| Groq | 번역 + 요약 | O (분당 30 요청) |
| Cohere | 번역 + 요약 | O (Trial) |
| DeepL | 번역 전용 | O (월 50만 자) |
| Naver Papago | 번역 전용 | O (일 1만 자) |
| Azure OpenAI | 번역 + 요약 | X |

자세한 API 키 발급 방법은 확장 프로그램 설정의 "가이드 보기" 링크를 클릭하세요.

## 사용 방법

1. **확장 프로그램 설정**: 설정(⚙️)에서 모델명과 API 키 입력 (AI 제공자는 자동 감지)
2. **스크립트 추출**: YouTube 영상 페이지에서 새로고침 버튼 클릭
3. **모드 선택**: 상단 탭에서 번역/일반/요약 모드 선택
4. **학습하기**: 스크립트 클릭으로 이동, 드래그로 선택
5. **저장하기**: 원하는 표현 저장
6. **보관함**: 저장된 표현 관리 및 복습

## 단축키

| 키 | 기능 |
|----|------|
| Space | 재생/일시정지 |
| ← / → | 5초 뒤로/앞으로 |

## 개인정보 처리방침

### 수집하는 정보
- **수집하지 않는 정보**: YouTube 시청 기록, 검색 기록, 개인 식별 정보
- **로컬 저장 정보**: 설정 값, API 키, 저장된 표현, 녹음 파일 (Chrome Storage API 사용)

### 데이터 처리
- **API 키**: 로컬 브라우저(Chrome Storage)에만 저장되며, 해당 AI 서비스 호출 시에만 사용됩니다.
- **번역/요약 텍스트**: 선택한 AI 서비스에만 전송되며, 본 확장 프로그램의 서버로 전송되지 않습니다.
- **저장된 표현 및 녹음**: 로컬 브라우저에만 저장되며 외부로 전송되지 않습니다.

### 제3자 서비스
번역 및 요약 기능 사용 시 선택한 AI 서비스(Google, OpenAI, Anthropic 등)로 텍스트가 전송됩니다. 각 서비스의 개인정보 처리방침을 참고하세요.

## 주의사항

- YouTube의 DOM 구조가 변경될 경우 스크립트 추출이 안 될 수 있습니다.
- 스크립트(자막)가 없는 영상에서는 작동하지 않습니다.
- API 사용량에 따라 비용이 발생할 수 있습니다. 각 서비스의 요금제를 확인하세요.
